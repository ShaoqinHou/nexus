# Customer UI Kit

Pixel-level recreation of the Nexus QR-scan ordering experience for diners.

Based on `packages/web/src/apps/ordering/customer/{CustomerApp,MenuBrowse,CartSidebar}.tsx`.

Layout: left category rail (desktop) / pill scroller (mobile) + main menu grid + sticky cart sidebar on the right. Click categories to jump; tap + on an item to add it; watch the cart update live.

Components: `CategoryRail`, `FeaturedStrip`, `MenuItemCard`, `CartSidebar`, `FloatingActions`, `TableChip`.
