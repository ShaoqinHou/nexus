# Merchant UI Kit

Pixel-level recreation of the Nexus staff console shell.

Based on `packages/web/src/platform/layout/PlatformShell.tsx` and surrounding merchant screens (Orders, Kitchen, Menu, Analytics). 

Components: `Sidebar`, `TopBar`, `NavItem`, `OrderCard`, `StatCard`, `OrdersBoard`.

Open `index.html` to view the interactive demo — click nav items to switch between Orders / Kitchen / Analytics panels.
